# POC - IMPTOX

The goal of this application, is to facilitate the usage of Labelbox and Google Cloud Storage in order to create an Assisted Annotation (Model Assisted Labeling in Labelbox) to accelerate the annotation of particle images. 

## Getting Started

Install the requirements in a conda environment, execute the main.py file. 

### Prerequisites

Anaconda
Labelbox account and API key
Google Cloud Service bucket
